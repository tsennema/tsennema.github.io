Game Room Navigator

There has been a recent trend in videos and streams around playing games that have had the map transitions randomized. This can become overwhelming and so this project is intended to help manage the process of playing through such a challenge. The default set-up is for Nintendo’s Pokémon Emerald game, but could be easily altered to manage other games.

Key Features Include:

  • Adding connections between room transitions
  • Labeling dead ends and points of interest
  • Provide navigation instructions between hubs or points of interest

Skills Showcased Include:

  • HTML/CSS/JavaScript FrontEnd Development
  • Using code for practical solutions
